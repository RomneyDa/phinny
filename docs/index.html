Phinny

An unofficial SimpleFIN viewer for macOS. It syncs your bank data into native charts, stored privately on your Mac.

Download for macOS View on GitHub
  • Native charts. Income vs. spending, net cash flow, and top spending, drawn with Swift Charts.
  • Private by design. Data lives in a local SQLite file; bank credentials stay in the macOS Keychain.
  • Any bank. SimpleFIN connects to thousands of institutions with read-only access.
  • Apple Card import. Bring in the CSV or OFX file you export from the iPhone Wallet app.
  • Try it first. A built-in demo loads sample data so you can explore before connecting.
Phinny is a Dallin Romney project. Not affiliated with SimpleFIN.
